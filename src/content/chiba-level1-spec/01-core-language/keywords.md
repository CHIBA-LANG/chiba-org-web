# 关键字集合

## 语法

关键字是在词法层具有保留意义的一组标识符，例如：

- `def`
- `type`
- `data`
- `union`
- `if`
- `else`
- `let`
- `unique`
- `send`
- `match`
- `for`
- `reset`
- `shift`
- `private`
- `unsafe`
- `true`
- `false`
- `via` for level2
- `as`
- TODO 需要补充

## 语义

关键字在关键字位置不能被当作普通标识符使用。

关键字集合决定 parser 的起始分支与某些歧义消解策略。
